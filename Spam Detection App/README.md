A simple machine learning powered spam detection tool. 
Frontend is next.js, backend is fastapi.

Requirements: Python, PIP & NPM
Using a virtual python environment is recommended to simplify package management

To run the frontend, CD to frontend/cos300049-frontend
then run npm install to install the dependencies.
Once installed, run the command "npm run dev" To run the application.

To run the backend, CD to backend
Then run pip install -r requirements.txt to install dependencies
then to run the application, run "uvicorn main:app --port 8000

Both applications should now both be running.
To view the frontend, navigate in a web browser to localhost:3000.
